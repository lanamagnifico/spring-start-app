# Spring Boot web-application for development
---
## Project includes:

- JSP views
- CSS Bootstrap
- Spring Data JPA
- H2 file DataBase
- User entity
- Rest services

*First initialization DataBase:*

*1) set property:*
    ```
    spring.datasource.initialization-mode=always
    ```

*2) put insert-SQL  in /resources folder with file-name data-h2.sql*
